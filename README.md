# Music App 🎶

Aplicación de música, construida con ReactJS

# Caracteristicas

-   Búsqueda en vivo
-   Fácil personalización de listas de reproducción
-   Fácil de crear o eliminar listas de reproducción
-   Fácil de agregar canciones a la lista de reproducción

# Uso de :

-   JavaScript 📐
-   React ⚛️
-   Hooks ⚓
-   Redux ⚓
-   Redux Saga ⚓
-   Styled-Components 🔳
-   Spotify API 🎶
-   Netlify 🚀
